# Arduino UNO Projects

## Overview

This repository contains a collection of beginner embedded systems projects developed using the Arduino UNO platform.

## Projects Included

### Traffic Light Controller
- Simulates a real-world traffic signal system
- Uses digital output pins and timing control

### LED Fading
- Demonstrates PWM (Pulse Width Modulation)
- Controls LED brightness smoothly

### Car Reverse Assistance System
- Uses an HC-SR04 ultrasonic sensor
- Measures obstacle distance
- Activates a buzzer when obstacles are detected

## Hardware Used

- Arduino UNO
- LEDs
- HC-SR04 Ultrasonic Sensor
- Buzzer
- Breadboard and jumper wires

## Skills Learned

- Embedded programming
- Sensor interfacing
- PWM control
- Digital I/O
